# ui-ux-pro-max

## Propósito

Sistema de diseño completo con 161 paletas de color por industria, 57 combinaciones tipográficas, y más de 50 estilos de layout y componentes. Es la biblioteca de referencia de diseño del proyecto: cuando hay que elegir una dirección visual, este skill provee opciones concretas y justificadas, no opciones genéricas.

---

## Cuándo activarse

- Se está eligiendo la paleta de colores para un proyecto nuevo.
- El usuario no sabe qué fuentes usar o quiere una sugerencia profesional.
- Se necesita decidir el estilo general: minimalista, editorial, corporativo, cálido, etc.
- El diseño actual no tiene personalidad o no corresponde a la industria del cliente.
- El usuario pide "hacer el diseño más profesional" o "que se vea de [industria]".

### 50+ ESTILOS DE LAYOUT

#### Hero layouts
- **Centrado clásico:** texto centrado, CTA centrado, imagen de fondo o abajo
- **Split 50/50:** texto izquierda, imagen derecha
- **Split 60/40:** texto dominante izquierda, imagen secundaria derecha
- **Full-width con overlay:** imagen de fondo, texto sobre overlay oscuro
- **Minimalista:** solo texto, sin imagen, mucho whitespace
- **Bold tipográfico:** un H1 muy grande, sin imagen, tipografía como protagonista

#### Feature layouts
- **Grid 3 columnas:** iconos + título + descripción
- **Grid 2 columnas asimétrico:** tarjeta grande izquierda + 2 pequeñas derecha
- **Lista horizontal:** features en fila con separadores
- **Alternado:** imagen izquierda/texto derecha, alternando por sección
- **Card grid:** tarjetas con hover, shadow y borde

#### Pricing layouts
- **3 columnas con destacado central:** el plan recomendado tiene borde o fondo diferente
- **2 opciones:** simple, para negocios con 2 planes
- **Toggle mensual/anual:** con switch de React
- **Lista de features comparativa:** tabla con checkmarks

#### Testimonials layouts
- **Carousel:** un testimonio a la vez, flechas de navegación
- **Grid 3 columnas:** todos visibles, sin interacción
- **Feature testimonial:** uno grande + 2 pequeños al costado
- **Quote wall:** testimonios cortos en masonry o bento
- **Video testimonials:** thumbnail con play button

#### Contact layouts
- **Formulario centrado:** formulario solo, sin información extra
- **Split:** formulario izquierda + información de contacto derecha
- **Minimalista:** solo email y WhatsApp, sin formulario
- **Map + form:** mapa de Google a la izquierda, formulario a la derecha
